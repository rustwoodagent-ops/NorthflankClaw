A daily email digest that reads your unread emails, summarizes them with AI, and delivers a formatted report to your inbox every morning.

## Chapter 1: The 3-Minute Briefing

### What Is an AI Workflow?

A sequence where AI processes information and takes action automatically.

**The Pattern:**
1. **Trigger** → Something happens (timer, new email, form submitted)
2. **Process** → AI reads and understands
3. **Action** → Something useful happens (summary sent, task created, reply drafted)

**Example you already use:** Spam filtering
- Trigger: Email arrives
- Process: AI analyzes content
- Action: Inbox or spam folder

### What You're Building

```
[7 AM Daily] → [Fetch unread emails] → [AI summarizes each] → [Send digest]
```

**What it does:**
- Runs automatically every morning
- Reads unread emails from last 24 hours
- Uses AI to summarize key points
- Emails you a single digest with everything

## Chapter 2: Tool Setup (30 Minutes)

### You Need 3 Free Tools

| Tool | Purpose | Cost |
|------|---------|------|
| **n8n** | Workflow engine (connects everything) | Free (self-hosted) |
| **OpenAI API** | The AI brain | ~$0.01 per email |
| **Gmail** | Email source and delivery | Free |

### Step 1: OpenAI Account (10 min)

1. Go to [platform.openai.com](https://platform.openai.com)
2. Sign up → verify email
3. Add payment method (Settings → Billing)
4. **Set $5 usage limit** (Settings → Billing → Usage limits)
5. Create API key (API keys → Create new secret key)
6. **Copy and save it**—you won't see it again

### Step 2: n8n Setup (15 min)

**Option A: Cloud (Easiest, 14-day trial)**
- Go to [n8n.io/cloud](https://n8n.io/cloud)
- Sign up → instance ready in minutes

**Option B: Your Computer (Free forever)**
1. Install [Docker Desktop](https://docker.com)
2. Run: `docker run -it --rm -p 5678:5678 -v ~/.n8n:/home/node/.n8n n8nio/n8n`
3. Open: `http://localhost:5678`

**Option C: Raspberry Pi/Old Computer (24/7 running)**
- Same Docker command as Option B
- Keeps running even when your main computer is off

### Step 3: Connect Credentials (5 min)

**Gmail:**
1. In n8n → Credentials → New → "Gmail OAuth2"
2. Follow prompts to authorize

**OpenAI:**
1. Credentials → New → "OpenAI"
2. Paste your API key

## Chapter 3: Build Your Workflow (60–90 Minutes)

### The 7-Step Process

#### Step 1: Create Workflow
1. In n8n, click **"Add Workflow"**
2. Name it: "Daily Email Digest"

#### Step 2: Schedule Trigger
1. Add node: **Schedule Trigger**
2. Set: Daily at 7:00 AM
3. Click **"Test Step"** to verify

#### Step 3: Fetch Emails
1. Add node: **Gmail → Get Many Messages**
2. Connect to trigger
3. Settings:
   - Credential: Your Gmail
   - Filter: Unread
   - After: 1 day ago
   - Max results: 20
4. Click **"Test Step"**

#### Step 4: AI Summarization
1. Add node: **OpenAI → Message a Model**
2. Connect to Gmail node
3. Settings:
   - Credential: Your OpenAI
   - Model: `gpt-4o-mini` (fast, cheap)
   - Message (User):

```
Summarize this email in 2-3 sentences. Identify if it requires a response or action.

From: {{ $json.From }}
Subject: {{ $json.Subject }}
Body: {{ $json.Text }}
```

4. Click **"Test Step"**

#### Step 5: Format Output
1. Add node: **Set**
2. Add field: `formatted_summary`
3. Value:

```
**From:** {{ $json.From }}
**Subject:** {{ $json.Subject }}
**Summary:** {{ $json.choices[0].message.content }}

---
```

#### Step 6: Combine Summaries
1. Add node: **Aggregate**
2. Set: "All items into a single item"
3. Field to aggregate: `formatted_summary`

#### Step 7: Send Digest
1. Add node: **Gmail → Send**
2. Settings:
   - To: your-email@gmail.com
   - Subject: "Your Daily Email Digest"
   - Text:

```
Good morning! Here are your unread emails from the last 24 hours:

{{ $json.formatted_summary }}

---
Generated by your AI workflow
```

### Final Step: Activate
1. Click **Save** (top right)
2. Toggle **Active** switch

**Done.** Your workflow runs every morning at 7 AM.

## Chapter 4: Test and Refine (20 Minutes)

### Quick Tests

| Test | Action | Expected Result |
|------|--------|-----------------|
| **Manual** | Click "Test Workflow" | Digest email arrives |
| **Real emails** | Mark 3 emails unread → Test | Those 3 summarized |
| **Empty inbox** | No unread emails → Test | Workflow completes, empty digest sent |

### Improve Your Summaries

**Too long?** Change prompt:
```
Summarize in 1 sentence maximum. Be concise.
```

**Missing details?** Change prompt:
```
Summarize including: sender, main point, any deadlines, action needed.
```

**Want bullet points?** Change prompt:
```
Summarize in 3 bullets: Who it's from, What they want, What you should do.
```

### Add Smart Filtering

Only summarize important emails:
1. Between Gmail and OpenAI nodes, add **IF** node
2. Condition: `{{ $json.From }}` contains "boss@" OR "client@" OR "urgent"
3. Connect "true" to OpenAI node
4. Leave "false" disconnected (emails skipped)

## Chapter 5: Troubleshooting

| Problem | Solution |
|---------|----------|
| "No credentials found" | Check Credentials section; select correct credential in each node |
| "API key invalid" | Verify key has no extra spaces; check payment method added; try new key |
| No emails found | Remove "Unread" filter temporarily to test; check date range |
| Poor summaries | Refine prompt; try `gpt-4o` instead of `gpt-4o-mini` |
| No digest received | Check spam folder; verify "To" address; check execution logs |
| Rate limit error | Add **Wait** node (1-2 seconds) between emails |
| Costs too high | Use `gpt-4o-mini`; limit to 10 emails; add filtering |

**Stuck?**
- n8n docs: [docs.n8n.io](https://docs.n8n.io)
- Community: [community.n8n.io](https://community.n8n.io)
