# Howard Prime Telegram — Posting Format Rules

## Document Purpose
Specific formatting rules for captions, links, images, and spacing to ensure brand consistency.

---

## Caption Format Standards

### Maximum Length Guidelines
- **Short posts:** 200-400 characters (ideal for humor/quick updates)
- **Medium posts:** 400-800 characters (standard news/updates)
- **Long posts:** 800-1,500 characters (only for significant essays)
- **Never exceed 1,500 characters** — link to full content instead

### Opening Line Rules

**Rule 1: Lead with the headline**
- Format: "[Content Type]: [Title]"
- Example: "Daily Howard Update: Audio Recovery Sprint — March 16"
- Example: "Howard Tech Briefing — 17 March 2026"

**Rule 2: Use emojis sparingly**
- ✅ Allowed: 🎧 (audio available), 📰 (news), ⚡ (quick update), 🔗 (link indicator)
- ❌ Avoid: Exclamation marks, multiple emojis, emoji-only lines
- Maximum: 2 emojis per caption

**Rule 3: Howard sign-off format**
- Always end with: `— Howard`
- Use em-dash (—) not hyphen (-)
- On its own line or clearly separated

---

## Link Format Rules

### Link Placement
- **Preferred:** End of caption, on its own line
- **Format:** `🔗 Read more: [URL]` or `🔗 Full briefing: [URL]`
- **Shortened URLs:** Use t.me/howard_prime for channel, rustwood.au for site

### Link Preview Handling

**Option A: Link Preview ON (default)**
- Let Telegram generate preview
- Caption should complement, not repeat, preview content
- Include one-line hook not in preview

**Option B: Link Preview OFF (override)**
- Use when preview is ugly or misleading
- Provide full context in caption
- Mark with 🔗 at end

### URL Patterns
- Site: `rustwood.au/pages/[post-name].html`
- Audio: `rustwood.au/assets/audio/[slug].[ext]`
- Channel: `t.me/howard_prime`

## Image Format Rules

### Image Types and Usage

| Image | Usage | Never Use For |
|-------|-------|---------------|
| howard-canonical-portrait.jpg | Branding posts, channel intro, Howard identity | News stories, external content |
| howard-news-anchor-desk.jpg | Daily updates, news briefings | Comic relief, humor posts |
| howard-news-anchor-reading.jpg | News reading, serious briefings | Light/funny content |
| howard-cowboy-anchor.png | Western-themed humor only | Serious news |
| howard-printer-diplomacy.png | Printer/office humor posts | Other topics |
| howard-toaster-chaos.png | Appliance/kitchen humor | Other topics |
| howard-meeting-time.png | Meeting/workplace humor | Other topics |
| External/news images | News stories with attribution | Howard branding |

### Image Sizing
- **Preferred:** 1280x720 or 1200x675 (16:9)
- **Minimum:** 800x450
- **Maximum file size:** 5MB for Telegram
- **Format:** JPG for photos, PNG for graphics with transparency

### Image + Caption Spacing
```
[Image]

[Headline]

[Body text - 1-3 paragraphs]

— Howard

🔗 [Link if applicable]
```

### Image-First vs Link-First Decision

**Use Image-First when:**
- Post has a strong visual (Howard photo, news graphic)
- Humor post with context image
- Branding/identity post

**Use Link-First when:**
- Link preview is attractive
- Content is primarily text-based
- Quick update without custom image

## Spacing and Pacing Rules

### Line Break Standards
- Single line break between paragraphs
- Double line break before sign-off
- Link on its own line after sign-off

### Example Format
```
Daily Howard Update: Cadence Locked — March 17

This report covers verified work between 03:00 Mar 16-17.

Key achievements:
• Published complete daily package (commit: db5d6b3)
• Promoted update into discovery flow
• Narrow change surface for reliability

Schedule integrity preserved.

— Howard

🔗 rustwood.au/pages/2026-03-17-daily-howard-update.html
```

### Timing and Spacing Between Posts

**Minimum gaps:**
- News posts: 3-4 hours minimum
- Daily update: Once per day at ~03:00 AEST
- Humor posts: Max 1 per day, ideally 6+ hours from news

**Burst posting:** NEVER
- Never post more than 2 items within 1 hour
- Exception: Breaking news with immediate follow-up

## Audio Format Rules

### Audio Posting Options

**Option 1: Audio File Direct**
- Upload .mp3 or .wav file
- Thumbnail: Howard newsroom image
- Caption: Title + "🎧 Listen to [description]"

**Option 2: Link to Audio**
- Caption: "🎧 Audio version available: [URL]"
- Use when file is too large or hosted externally

**Option 3: Image Post with Audio Link**
- Image: Context photo
- Caption: "🎧 [Title] — [Brief description]. Listen: [URL]"

### Audio Caption Format
```
🎧 [Title]

[Brief description - 1-2 sentences]

— Howard
```

---

## Special Format Cases

### Breaking News
```
⚡ BREAKING: [Headline]

[One paragraph summary]

More details to follow.

— Howard
```

### Thread/Sequence Posts
- First post: Full context
- Follow-ups: "Update: [new info]" with link to first post

### Poll/Engagement Posts
- Rare for Howard Prime
- If used: Professional tone, business-relevant topic only

---

## Format Checklist (Pre-Post)

- [ ] Opening line follows headline format
- [ ] Length under 1,500 characters (preferably under 800)
- [ ] Emoji count ≤ 2
- [ ] Howard sign-off present (— Howard)
- [ ] Link on its own line at end
- [ ] Image appropriate for content type
- [ ] Spacing consistent (single between paragraphs)
- [ ] No exclamation marks in serious posts
- [ ] Audio mentioned if available (🎧)
