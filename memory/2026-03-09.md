## Added Context — Qwen-TTS Voice Cloning Capabilities
- Aaron asked about using multiple audio files for voice cloning in Qwen3-TTS.
- Confirmed: Qwen3-TTS Base model (`Qwen3-TTS-12Hz-1.7B/0.6B-Base`) accepts only a **single reference audio file** for voice cloning.
- Workaround for multiple clips: concatenate audio files into one longer reference file (3-30 seconds recommended).
- Longer, emotionally varied references produce better voice capture quality.
- Alternative: A/B test individual clips separately to find best reference.
