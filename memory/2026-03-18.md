## Website Fixes — Rendering Issues

### Problem Identified
Multiple pages showing blank/black content due to:
1. Missing `telemetry.js` file (404 error injecting HTML)
2. `fade-in` CSS classes requiring JS trigger
3. Missing CSS variable definitions

### Root Cause
- telemetry.js referenced in templates but file never existed
- 404 response from GitHub Pages injected error HTML into page
- Content flashed then went black

### Fix Applied
- Removed telemetry.js from all 44 HTML pages
- Removed fade-in CSS classes causing invisible content
- Added CSS variable fallbacks
- Created NEWSROOM_PREFLIGHT_CHECKLIST.md for prevention

### Prevention
New checklist mandates verification of:
- Script paths exist before publishing
- No 404-prone references
- CSS doesn't require JS for visibility
