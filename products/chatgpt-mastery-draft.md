## Module 1: The One Setting That Changes Everything (5 minutes)

### Custom Instructions

Most people ignore this feature. This is a mistake.

**What it does:** Tells ChatGPT who you are and how you want responses — automatically, every time.

**How to set it up:**

1. Click your name (bottom left)
2. Select "Custom instructions"
3. Fill in: "What would you like ChatGPT to know about you?"

**Example for a business owner:**
```
I run a small e-commerce business selling handmade jewelry.
My customers are women aged 30-50 who value craftsmanship.
I need clear, actionable advice I can implement quickly.
```

**Example for a consultant:**
```
I'm a management consultant working with mid-sized companies.
I need professional, structured communication.
My clients value clarity over cleverness.
```

4. Fill in: "How would you like ChatGPT to respond?"

**Example instructions:**
```
- Be concise. Get to the point quickly.
- Use bullet points for lists.
- When giving advice, explain why it works.
- Avoid jargon unless I ask for technical details.
```

**Why this matters:** Without this, ChatGPT treats you like a generic user. With it, every response is tailored to your context.

## Module 2: The Template Method (10 minutes)

### Why Most Prompts Fail

**Weak prompt:** "Write an email about my new product"

**Result:** Generic, bland, unusable.

**Strong prompt structure:**
```
CONTEXT: [Who you are, situation]
TASK: [Exactly what you need]
FORMAT: [How it should look]
TONE: [How it should sound]
CONSTRAINTS: [Limitations/requirements]
```

**Example:**
```
CONTEXT: I'm a freelance designer announcing a pricing increase to long-term clients.
TASK: Write an email explaining the increase while maintaining goodwill.
FORMAT: Short email, 150 words max.
TONE: Professional but warm, confident but not defensive.
CONSTRAINTS: Mention the increase is effective next month. Include a thank you for their loyalty.
```

**Result:** Usable first draft, specific to your situation.

### 25 Ready-to-Use Templates

**Template 1: The Rewrite**
```
Rewrite this [email/document/message] to be [clearer/shorter/more professional].
Original: [paste text]
Target audience: [description]
```

**Template 2: The Comparison**
```
Compare [Option A] vs [Option B] for [specific use case].
I care about: [factors that matter to you]
Format: Pros/cons list with recommendation.
```

**Template 3: The Explainer**
```
Explain [topic] like I'm [target audience].
Use an analogy if helpful.
Keep it under 200 words.
```

**Template 4: The Decision Helper**
```
I'm deciding between [Option A] and [Option B].
Context: [relevant details]
My priorities: [what matters most]
What should I consider that I might be missing?
```

**Template 5: The Outline Creator**
```
Create an outline for [document type: blog post/presentation/report] about [topic].
Target length: [word count or minutes]
Key points to cover: [bullet list]
```

## Module 3: The Follow-Up Habit (10 minutes)

### Your First Draft Is Rarely Your Best

**The mistake:** Accepting ChatGPT's first response.

**The fix:** Treat it as a starting point, not a finished product.

### Effective Follow-Up Patterns

**"Make it shorter"**
```
Cut this by 50%. Keep the key points.
```

**"Add examples"**
```
Add 2-3 specific examples to illustrate the main points.
```

**"Change the tone"**
```
Make this more [formal/casual/direct/encouraging].
```

**"Simplify"**
```
Explain this to someone with no background in [topic].
```

**"Expand on one point"**
```
Go deeper on [specific point]. The rest is fine.
```

### The Iteration Rule

Good output usually takes 2-3 exchanges. Not 10. Not 1.

## Module 4: When ChatGPT Gets It Wrong (5 minutes)

### Hallucination Recognition

**Red flags:**
- Specific facts that seem generic
- Confident statements you can't verify
- References to sources you didn't provide

**The fix:**
- Ask for sources: "What are you basing this on?"
- Verify externally before acting
- Use ChatGPT for structure, not facts you don't know

### Pushing Back

If the output isn't right, say so:

```
This isn't what I was looking for. Let me clarify:
[specific correction]
Let's try again with that in mind.
```
