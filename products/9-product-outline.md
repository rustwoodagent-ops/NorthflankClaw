## The 10 Workflows

1. **Morning Briefing Generator**
   - Aggregates calendar + email + news
   - Delivers concise morning summary
   - Tools: Gmail, Calendar, OpenAI

2. **Content Repurposer**
   - Takes one long-form piece
   - Creates: LinkedIn post, Twitter thread, newsletter excerpt
   - Tools: Google Docs, OpenAI

3. **Meeting Prep Assistant**
   - Reads calendar event + attendee LinkedIn profiles
   - Generates talking points and questions
   - Tools: Calendar, LinkedIn scraping, OpenAI

4. **Customer Inquiry Router**
   - Reads incoming email
   - Classifies: urgent, sales, support, spam
   - Routes to appropriate person/system
   - Tools: Gmail, OpenAI, Slack/Teams

5. **Weekly Report Compiler**
   - Gathers data from multiple sources
   - Formats executive summary
   - Sends to stakeholders
   - Tools: Sheets/Notion, OpenAI, Email

6. **Social Media Scheduler**
   - Drafts posts from content calendar
   - Suggests optimal posting times
   - Queues to Buffer/Hootsuite
   - Tools: Sheets, OpenAI, Buffer API

7. **Invoice Follow-up**
   - Tracks unpaid invoices
   - Sends polite reminder sequences
   - Flags overdue for human follow-up
   - Tools: Sheets/Accounting software, Gmail

8. **Research Synthesizer**
   - Takes 5-10 articles/links
   - Extracts key points
   - Creates summary with citations
   - Tools: Web fetch, OpenAI

9. **Onboarding Sequence**
   - Triggers on new client signup
   - Sends personalized welcome series
   - Schedules check-in calls
   - Tools: Email, Calendar, CRM

10. **Error Alert Summarizer**
    - Receives system alerts
    - Translates technical errors to plain English
    - Suggests priority and action
    - Tools: Webhook, OpenAI, Slack
